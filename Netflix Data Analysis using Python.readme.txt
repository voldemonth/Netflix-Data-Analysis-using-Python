Netflix Data Analysis using Python

This project analyzes the Netflix dataset using Python to understand patterns in movies and TV shows available on the platform. The analysis includes data cleaning, exploratory data analysis (EDA), visualizations, and insights.

The goal of this project is to explore Netflix content and identify trends such as the distribution of movies and TV shows, top countries producing content, content ratings, and other important insights.

Tools Used

Python

Pandas

Matplotlib

Jupyter Notebook

Project Steps
1. Data Cleaning

Handled missing values

Fixed data types

Cleaned columns such as director, cast, country, and date_added

2. Exploratory Data Analysis (EDA)

Several analysis questions were explored, such as:

Movies vs TV Shows distribution

Top countries producing Netflix content

Most common ratings

Content trend by release year

Top directors with the most titles

Average movie duration

Longest movies on Netflix

3. Data Visualization

Visualizations were created to better understand the data, including:

Movies vs TV Shows distribution

Top countries producing Netflix content

Ratings distribution

Content trend by year

Top directors with the most titles

Key Insights

Movies are more common than TV shows on Netflix.

The United States produces the highest number of titles.

TV-MA is the most common content rating.

Netflix content increased significantly after 2015.

Some director information is missing in the dataset.

Conclusion

The analysis shows that Netflix contains a large collection of movies and content from multiple countries, with the United States contributing the most titles. The platform has expanded its content significantly in recent years.
